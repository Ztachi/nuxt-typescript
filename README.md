# 项目
## 朴新在线PC学生端官网

# 描述
## 朴新在线PC学生端官网服务器端渲染版本。以nuxt为前端基本框架，elementUI为UI框架。koa为服务器端框架，pm2进程守护

# 环境
推荐安装nodejs**12.16.1**版本
